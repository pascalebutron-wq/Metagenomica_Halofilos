# Pipeline assembly utilizado para LV8T

Este folder contiene los comandos y pasos utilizados para:

- Trimming (Trim Galore)
- Assembly (SPAdes / MEGAHIT)
- Evaluación (QUAST)
- Anotación (Prodigal + EggNOG-mapper)
- Taxonomía (Kraken2 / Kaiju)
- (Opcional) Mapping y visualización (Bowtie2 + Samtools + IGV)
- (Opcional) Descarga de genomas y árbol
